    ## Endpoints de l'API

### /sending/upload

#### Description
Permet de télécharger des fichiers vers le serveur.

#### Réponses

- **201 Created**
  - **Message** : Vos fichiers sont envoyés avec succès.
  - **Exemple de réponse** :
    ```json
    {
      "message": "Vos fichiers sont envoyés avec succès.",
      "fileId": "abc123"
    }
    ```

- **403 Forbidden**
  - **Message** : Vous n'êtes pas autorisé à envoyer plus de 2Go.
  - **Exemple de réponse** :
    ```json
    {
      "error": "Vous n'êtes pas autorisé à envoyer plus de 2Go."
    }
    ```

- **415 Unsupported Media Type**
  - **Message** : Ce type de fichier n'est pas supporté.
  - **Exemple de réponse** :
    ```json
    {
      "error": "Ce type de fichier n'est pas supporté."
    }
    ```

- **500 Internal Server Error**
  - **Message** : Une erreur interne au serveur est survenue.
  - **Exemple de réponse** :
    ```json
    {
      "error": "Une erreur interne au serveur est survenue."
    }
    ```

### /sending/{id}

#### Description
Permet de récupérer les détails d'un envoi spécifique.

#### Réponses

- **200 OK**
  - **Message** : Détails de l'envoi.
  - **Exemple de réponse** :
    ```json
    {
      "id": "abc123",
      "status": "completed",
      "fileSize": "1.5GB",
      "uploadedAt": "2024-06-27T14:35:22Z"
    }
    ```

- **404 Not Found**
  - **Message** : La ressource n'est pas trouvée.
  - **Exemple de réponse** :
    ```json
    {
      "error": "La ressource n'est pas trouvée."
    }
    ```

- **500 Internal Server Error**
  - **Message** : Une erreur interne au serveur est survenue.
  - **Exemple de réponse** :
    ```json
    {
      "error": "Une erreur interne au serveur est survenue."
    }
    ```
